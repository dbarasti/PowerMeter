# Porta Modbus
# IMPORTANTE: Usa il path del dispositivo DENTRO il container Docker, non quello del sistema host!
#
# Windows/WSL2:
#   - COM1 -> /dev/ttyS0
#   - COM2 -> /dev/ttyS1
#   - COM3 -> /dev/ttyS2
#   - COM4 -> /dev/ttyUSB0 (mappato da /dev/ttyS3 nel docker-compose.yml)
#
# Linux:
#   - /dev/ttyUSB0 o /dev/ttyACM0
#
# macOS:
#   - Non supportato direttamente, vedi DOCKER_README.md
MODBUS_PORT=/dev/ttyUSB0
